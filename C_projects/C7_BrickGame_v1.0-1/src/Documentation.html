About the game

Эта игра разработана на языке C и представляет собой консольную версию легендарного Тетриса.

Основная библиотека для отрисовки — ncurses, которая идеально подходит для создания текстовых пользовательских интерфейсов в терминале, что делает её отличным выбором для ретро-игр, таких как Тетрис.
Игровой процесс классический: тетромино падают сверху игрового поля, и задача игрока — укладывать их так, чтобы полностью заполнять горизонтальные линии. Заполненные линии исчезают, принося игроку очки. По мере игры скорость падения фигур увеличивается, усложняя процесс.
Основные задачи разработки включали корректное отображение падающих фигур, управление их движением и поворотами, а также реализацию удаления заполненных линий и подсчета очков. Библиотека ncurses значительно упростила обработку пользовательского ввода, отображение фигур в консоли и работу с цветами и символами для создания интерфейса игры.
Кроме того, реализована проверка корректности положения фигур, чтобы предотвратить их выход за пределы поля и корректно обрабатывать столкновения.
